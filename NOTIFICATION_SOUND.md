# Своя мелодия для напоминаний

Приложение поддерживает **собственный звук** для всех напоминаний (задачи, ежедневные, просрочка).

**Беззвучный режим и «Не беспокоить»:** если у пользователя включён беззвучный режим, тихий режим или «Не беспокоить», звук уведомлений (в том числе ваша мелодия) **не воспроизводится** — так решает система Android/iOS, приложение это не переопределяет. Это нормальное поведение и соответствует настройкам пользователя.

## Имя файла

В коде задано имя: **`reminder.wav`**.  
Чтобы использовать другую мелодию, замените константу `NOTIFICATION_SOUND` в `src/lib/notifications.js` и положите файл с тем же именем в нужные папки.

## Куда положить файл

### Android

Папка: **`frontend/android/app/src/main/res/raw/`**

- Положите файл **`reminder.wav`** в эту папку.
- Имя файла: только **латиница**, **нижний регистр** (ограничение Android для ресурсов).
- Рекомендуемый формат: **WAV**; длительность до ~30 секунд.

Если папки `raw` нет — создайте её рядом с `drawable` и `values`.

### iOS

1. Откройте проект **`frontend/ios/App/App.xcodeproj`** в Xcode.
2. Добавьте файл **`reminder.wav`** в проект (правый клик по группе App → Add Files to "App"…).
3. Убедитесь, что в диалоге отмечено **"Copy items if needed"** и в списке целевых таргетов отмечен **App**.
4. Файл должен оказаться в бандле приложения (проверьте в Build Phases → Copy Bundle Resources).

Рекомендуемый формат: **WAV** или **AAC**; длительность до ~30 секунд.

## Отключение своей мелодии

Чтобы снова использовать системный звук, в `src/lib/notifications.js` задайте:

```js
const NOTIFICATION_SOUND = ''
```

Пересоберите приложение и при необходимости переустановите (на Android 8+ звук канала может кэшироваться).
