# Что делать дальше — передача кода в Xcode через GitHub

Краткая последовательность: вы пушите проект в GitHub с Windows, на Mac клонируете репозиторий и собираете архив в Xcode для загрузки в App Store.

---

## 1. На Windows: отправить код в GitHub

### Если репозиторий ещё не создан

1. Откройте PowerShell в папке проекта:
   ```powershell
   cd c:\Users\PC\Desktop\projects\cursor\DeloApp
   ```

2. Инициализируйте Git и сделайте первый коммит:
   ```powershell
   git init
   git add .
   git commit -m "Дело: готово к сборке iOS"
   ```

3. На GitHub: https://github.com/new — создайте **новый репозиторий** (например, `DeloApp`).  
   Не добавляйте README, .gitignore и лицензию — у вас уже есть файлы.

4. Привяжите удалённый репозиторий и отправьте код (подставьте свой логин и имя репо):
   ```powershell
   git remote add origin https://github.com/ВАШ_ЛОГИН/DeloApp.git
   git branch -M main
   git push -u origin main
   ```

### Если репозиторий уже есть (как раньше отправляли через GitHub)

Просто запушьте текущие изменения:

```powershell
cd c:\Users\PC\Desktop\projects\cursor\DeloApp
git add .
git commit -m "Обновление для iOS: UILaunchScreen, документация"
git push origin main
```

---

## 2. На Mac: получить код и собрать в Xcode

1. **Клонировать** (один раз; если уже клонировали — переходите к шагу 2):
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/DeloApp.git
   cd DeloApp/frontend
   ```

2. **Установить зависимости и собрать веб-часть:**
   ```bash
   npm install
   npm run build
   npx cap sync ios
   ```

3. **Открыть проект в Xcode:**
   ```bash
   npx cap open ios
   ```
   Откроется Xcode с проектом `App` (workspace).

4. **В Xcode:**
   - Слева выберите проект **App**.
   - Вкладка **Signing & Capabilities** — укажите ваш **Team** (Apple Developer).
   - Убедитесь, что **Bundle Identifier**: `app.delodelai`.
   - Вверху выберите схему **App** и устройство **Any iOS Device (arm64)**.
   - Меню **Product** → **Clean Build Folder**, затем **Product** → **Archive**.

5. **После успешной сборки архива:**
   - Откроется окно **Organizer**.
   - Выберите только что созданный архив.
   - Нажмите **Distribute App** → **App Store Connect** → **Upload** и пройдите мастер.

6. **В App Store Connect** (https://appstoreconnect.apple.com):
   - Откройте приложение «Дело» и нужную версию.
   - В блоке **Сборка** нажмите **+** и выберите загруженную сборку.
   - Заполните метаданные (тексты — в `ТЕКСТЫ_ДЛЯ_ВЫКЛАДКИ.md`, иконки и скриншоты — в `store-assets/`).
   - Отправьте на проверку.

---

## При следующих обновлениях

**На Windows:** после изменений в коде:
```powershell
git add .
git commit -m "Описание изменений"
git push origin main
```

**На Mac:** перед новой сборкой:
```bash
cd /путь/к/DeloApp
git pull origin main
cd frontend
npm run build
npx cap sync ios
npx cap open ios
```
Дальше в Xcode снова **Product** → **Archive** и загрузка в App Store Connect.
