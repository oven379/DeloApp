## 1. Визуальные материалы (папка store-assets/)

Выполните `cd frontend && npm run store-assets`, затем загрузите:

### Google Play
| Что | Файл |
|-----|------|
| Значок | `store-assets/icon-512.png` |
| Картинка для описания | `store-assets/feature-graphic-1024x500.png` |
| Скриншоты телефона | `store-assets/phone/screenshot-1.png` … `screenshot-8.png` (минимум 2) |

### App Store
| Что | Файл |
|-----|------|
| Иконка 1024×1024 | `store-assets/icon-1024.png` |
| Скриншоты iPhone 6.5" | `store-assets/app-store/iphone-6.5/screenshot-1.png` … `screenshot-10.png` |

## 2. Тексты

Скопируйте из **ТЕКСТЫ_ДЛЯ_ВЫКЛАДКИ.md**.

### Google Play
- Название: Дело
- Краткое описание (80 симв.): Задачи на сегодня и завтра. Выгрузи дела из головы.
- Полное описание — см. файл
- Описание изменений: Первая версия приложения.

### App Store
- Название: Дело
- Подзаголовок: Задачи на сегодня и завтра
- Описание — см. файл
- Ключевые слова: задачи,напоминания,список дел,планер,офлайн,без рекламы
- Рекламный текст (по желанию): краткий слоган до 170 символов

## 4. Сборки

| Платформа | Где взять |
|-----------|-----------|
| AAB (Google Play) | Android Studio → Build → Generate Signed Bundle |
| IPA (App Store) | Xcode → Product → Archive |
