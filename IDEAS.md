# Обзор приложения «Дело» — пользовательский путь и предложения

## Что уже хорошо работает

- **Сегодня / Завтра** — переключение дня, добавление дел на завтра.
- **Добавление** — поле внизу, плейсхолдер зависит от выбранного дня.
- **Редактирование** — клик по тексту, textarea с полным текстом, Ctrl+Enter — сохранить.
- **Выполнение** — галочка, свайп вправо на мобильном.
- **Удаление** — корзина, опциональное подтверждение, свайп влево.
- **Перенос** — кнопка «Завтра» у дел на сегодня.
- **Напоминания** — по задаче (день + время) и два ежедневных слота в настройках (раз в день).
- **Настройки** — колокольчик → Тема (Светлая / Тёмная / Моя), режим списка, время уведомлений.
- **Поиск** — фильтр по тексту.
- **Пустой экран** — подсказка, когда нет дел на выбранный день.
- **Прогресс** — X/Y · Z осталось, полоска в шапке.

---

## Что добавить (для лучшего пользовательского пути)

| Предложение | Зачем |
|-------------|--------|
| **Подсказка при редактировании** | Под полем редактирования короткая строка: «Ctrl+Enter — сохранить». Сейчас неочевидно, как выйти из режима редактирования без клика вне поля. |
| **«Ничего не найдено» при поиске** | Если поиск включён и результатов 0 — показывать отдельное сообщение («По запросу ничего не найдено»), а не общий пустой экран. |
| **Размер шрифта в настройках** | В данных уже есть `fontSize` (small/medium/large), но в UI его нет. Либо добавить в настройки переключатель «Шрифт: Маленький / Средний / Большой», либо перестать хранить и применять. |
| **Подтверждение перед импортом** | Сейчас импорт (если вернёте кнопку) перезаписывает все дела без предупреждения. Имеет смысл: «Заменить все текущие дела? Да / Отмена». |

---

## Что убрать или упростить

| Предложение | Зачем |
|-------------|--------|
| **Удалить компонент ReminderDrum** | `ReminderDrum.jsx` и `ReminderDrum.css` нигде не импортируются — мёртвый код. Удаление упростит проект. |
| **Упростить fontSize** | Если не планируете UI для размера шрифта — убрать `fontSize` из настроек и вызовы `applyFontSize` в App, чтобы не хранить неиспользуемую опцию. |

## Опционально (по желанию)

- **Краткая подсказка при первом визите** — один раз показать туториал или подсказку («Добавляйте дела внизу, свайп вправо — выполнить» и т.п.).
- **Escape в поле ввода** — по Escape очищать поле ввода (сейчас только в редактировании задачи).
- **Вернуть экспорт/импорт в настройки** — одной строкой «Резервная копия: Экспорт · Импорт», если снова понадобится бэкап.
