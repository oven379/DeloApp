# Визуальные материалы для Google Play и App Store

Сгенерировано скриптом `frontend/scripts/make-store-assets.js`.

## Как обновить

```bash
cd frontend
npm run store-assets
```

---

## Google Play

| Файл | Где в консоли |
|------|---------------|
| **icon-512.png** | Значок приложения (512×512) |
| **feature-graphic-1024x500.png** | Картинка для описания |
| **phone/screenshot-*.png** | Скриншоты для смартфона (2–8 шт) |
| **tablet-7/**, **tablet-10/** | Планшеты (по желанию) |

## App Store

| Файл | Где в консоли |
|------|---------------|
| **icon-1024.png** | Иконка приложения (1024×1024, PNG, без прозрачности) |
| **app-store/iphone-6.5/screenshot-*.png** | Скриншоты для iPhone 6.5" (1242×2688 px, до 10 шт) |

Скриншоты в `app-store/iphone-6.5/` подходят для вкладки **iPhone** в разделе 6.5-дюймового дисплея. Тексты для App Store — в **ТЕКСТЫ_ДЛЯ_ВЫКЛАДКИ.md**.
