# Выкладка «Дело» в App Store

Краткая инструкция для публикации в Apple App Store.

---

## 1. Требования

- **Mac** с установленным Xcode
- **Apple Developer Program** ($99/год), аккаунт активен
- **App ID** зарегистрирован в Apple Developer с Bundle ID: `app.delodelai`

---

## 2. App Store Connect

1. Войдите: https://appstoreconnect.apple.com
2. **Мои приложения** → **«+»** → **Новое приложение**
3. Заполните:
   - **Платформы:** iOS
   - **Название:** Дело
   - **Основной язык:** Русский
   - **Bundle ID:** выберите `app.delodelai` (должен быть создан в Certificates, Identifiers & Profiles)
   - **SKU:** `delodelai-app` (любой уникальный)
   - **Пользовательский доступ:** Полный доступ

---

## 3. Сборка в Xcode

```bash
cd frontend
npm run build
npx cap sync ios
npx cap open ios
```

В Xcode:
1. Выберите **Team** (ваш Apple Developer аккаунт)
2. Убедитесь, что **Bundle Identifier** = `app.delodelai`
3. **Product** → **Archive**
4. После успешной сборки: **Distribute App** → **App Store Connect** → **Upload**

---

## 4. Контент в App Store Connect

| Поле | Значение |
|------|----------|
| **Подзаголовок** (до 30 симв.) | Задачи на сегодня и завтра |
| **Описание** | Текст из **ТЕКСТЫ_ДЛЯ_ВЫКЛАДКИ.md** |
| **Ключевые слова** | задачи,напоминания,список дел,планер,офлайн,без рекламы |
| **URL поддержки** | https://t.me/manager379team или https://delodelai.ru/ |
| **Политика конфиденциальности** | https://delodelai.ru/privacy/ |
| **Категория** | Productivity |
| **Скриншоты** | По размерам устройств (6.7", 6.5", 5.5" и т.д.) |
| **Иконка** | 1024×1024 px, PNG, без прозрачности |

---

## 5. Что нужно для Capabilities

Для напоминаний в приложении включите **Push Notifications** в Xcode:
- Проект → Signing & Capabilities → **+ Capability** → Push Notifications

(Локальные уведомления работают и без push-сервера, но capability может понадобиться.)

---

## 6. Чек-лист перед «Добавить для проверки»

Если кнопка «Добавить для проверки» недоступна, в App Store Connect нужно заполнить всё из списка ниже.

| Что нужно | Где заполнить | Значение / действие |
|-----------|----------------|---------------------|
| **URL политики конфиденциальности** | **Конфиденциальность приложения** (App Privacy) | Укажите: `https://delodelai.ru/privacy/` |
| **Скриншот iPad 12.9"/13"** | Версия 1.0 → **Снимки экрана** → iPad (13") | Загрузите снимки 2064×2752 px (или 2048×2732, 2732×2048). Сгенерировать: `npm run store-assets` → папка `store-assets/app-store/ipad-pro-13/` |
| **Основная категория** | **Информация о приложении** (App Information) | Выберите: **Productivity** (Производительность) |
| **Соответствие экспортным требованиям** | Сборка → **Управлять** (или при загрузке сборки) | Для этой сборки в проекте уже добавлен ключ `ITSAppUsesNonExemptEncryption = false` (только HTTPS). В форме укажите: приложение не использует шифрование сверх стандартного, или ответьте «Нет» на вопрос про шифрование. |
| **Ценовой ярус** | **Ценообразование** (Pricing) | Выберите **Бесплатно** (Free) или нужный ярус |
| **Вопросы о возрастных ограничениях** | **Информация о приложении** → Возрастной рейтинг | Пройдите опрос (для списка дел без контента 4+ или 12+ обычно подходит) |
| **Права на публикуемые материалы** | **Информация о приложении** | Отметьте, что у вас есть права на контент приложения (логотип, тексты и т.д.) |

После заполнения всех пунктов кнопка **«Добавить для проверки»** станет активной.

---

## 7. Отправка на проверку

1. В App Store Connect: **Тестовая информация** (если нужно)
2. **Версия** → **Добавить для проверки** → **Отправить на проверку**
3. Ожидание: обычно 1–3 дня

---

## Если при валидации: «Apps that support Multitasking on iPad must provide launch screen...»

В проекте уже настроено:
- **UILaunchStoryboardName** = `LaunchScreen` (storyboard)
- **UILaunchScreen** (словарь с `UIImageName` = `Splash`) — для iOS 14+

Оба ключа есть в `frontend/ios/App/App/Info.plist`. Если ошибка сохраняется, в Xcode проверьте, что в target **App** → **General** → **App Icons and Launch Screen** указан Launch Screen (Storyboard или Asset).
